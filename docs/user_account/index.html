Why get a user account?⌗

Having a user account on hack.courses allow you to login onto your own and private Linux account from anywhere.

The files (scripts, notes, code, programs) you will create in your private home directory will be persisted and kept as long as hack.courses and your account exists and have enough storage to accept them.

Oh, also, it’s free!

Who can get a user account and how?⌗

Any student can apply for a user account by sending an e-mail to contact AT fenrir.pro with the following subject : hack.courses user account and explaining in a few sentences what they intend to do with it and which username should be used!
